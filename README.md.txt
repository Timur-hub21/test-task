Тестовое задание реализовано на языке c#, на платформе Visual Stiduo 2019.

1 Открываем файл "test task itech.sln
2 Нажимаем F5 для запуска проекта
3 Вводим количество точек. Например 2 точки (1,3) (4,4)
где для первой координаты х=1 y=3, для второй х=4 y=4.
4 Получаем результат пути робота ENNNDEEEND и используемые координаты.
